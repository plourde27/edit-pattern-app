FizzBuzz
~~~
Write a function to solve the classic FizzBuzz example problem.
Given an integer N, return:
* FizzBuzz, if N is divisible by both 3 and 5
* Fizz, if N is divisible by 5, but not 3
* Buzz, if N is divisible by 3, but not 5
* an empty string, if N is divisible by neither 3 nor 5
